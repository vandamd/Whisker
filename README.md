<p>App to hide menubar app icons in macOS.</p>

![GitHub License](https://img.shields.io/github/license/vandamd/Whisker)
![GitHub Release](https://img.shields.io/github/v/release/vandamd/Whisker)

## Usage
CMD + Drag the circle menu icon to the right of the icons you want to hide. 

Hide via the menu button or toggle visiblity with your hotkey (default: CMD + SHIFT + Y).

## Installation
Install via Homebrew:
```brew install vandamd/tap/whisker```

Or download the latest release from the [releases page](https://github.com/vandamd/Whisker/releases/latest).
